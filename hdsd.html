Hướng dẫn khai báo LaTeX với \usepackage{hyperref}

Dưới đây là hướng dẫn chi tiết về cách khai báo và sử dụng các lệnh trong LaTeX với các gói hyperref, tcolorbox, và các lệnh tùy chỉnh.

        \usepackage{hyperref} % Gói dùng để tạo liên kết
        \usepackage{comment}  % Gói hỗ trợ ẩn các đoạn văn bản
        \usepackage{tcolorbox} % Gói để tạo khung đẹp
        \usepackage{tikz} % Gói hỗ trợ vẽ hình và tạo khung tùy chỉnh
        \usepackage{fontawesome} % Gói biểu tượng cho tiêu đề
        
        % Biến kiểm soát hiển thị lời giải
        \newif\ifshowloigiai
        \showloigiaitrue % Hiển thị lời giải, đổi thành \showloigiaifalse để ẩn
        
        % Định nghĩa lại lệnh \loigiai với khung đẹp
        \renewcommand{\loigiai}[1]{%
            \ifshowloigiai
            \begin{tcolorbox}[colback=orange!10, colframe=orange!85!red,
            title=\centering\faLightbulbO\ \textbf{Lời giải}]
            #1
            \end{tcolorbox}
            \fi
        }
        
        % Lệnh tạo liên kết với hộp màu cam
        \newcommand{\myfbox}[2][]{%
            \hypersetup{linkcolor=white}
            \hyperlink{#1}{\fcolorbox{orange!85!black}{orange!30!white}{%
            \textbf{\textcolor{black}{\color{blue}#2}}}}
        }
        
        % Lệnh tạo liên kết với hộp bo tròn
        \newcommand{\mfbox}[2][]{%
            \hypersetup{linkcolor=black}
            \hyperlink{#1}{\begin{tikzpicture}
            \node[fill=orange!30!white, rounded corners=10pt,
            text=blue, font=\bfseries] {#2};
            \end{tikzpicture}}
        }

Sử dụng \loigiai{} để chèn lời giải vào tài liệu, và \myfbox{}, \mfbox{} để tạo các hộp liên kết đẹp trong LaTeX. Bạn có thể dễ dàng thay đổi màu sắc và hình dạng của hộp.
